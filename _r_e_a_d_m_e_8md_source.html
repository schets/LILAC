1 LILAC: Learning and Integration of Lasers for Adaptive Control
2 
3 This project provides a framework with which to analyze/control lasers, and more generally, any tunable dynamical system.
4 
5 You can find documentation on the [wiki](http://github.com/schets/LILAC/wiki), and doxygen generated [advanced documentation](http://schets.github.io/LILAC) as well
6 
7 Installation:
8 This project depends on [FFTW](www.fftw.org), and the makefile has configurations for GCC and the Intel C++ Compiler. These are simple to install on Linux and Macintosh systems and each site provides installation instructions.
9 
10 The library Eigen3 is included with the source. This is done to simplify installation, especially on NERSC which is iffy about non-standard libraries. Eigen3 is a set of headers, so no work is required on the part of the user.
11 
12 
13 Installation on Linux:
14 
15 1. From the command line, proceed to the directory in which you want to have lilac installed.
16 2. Run the command: git clone https://github.com/schets/LILAC lilac
17 3. Enter the lilac directory (cd lilac)
18 4. Compile the code by running make
19 5. Upon successful compilation, the lilac binary can be found in the directory bin
20 
21 Or, from to installation directory, copy and paste this into the command line
22 
23 git clone https://github.com/schets/LILAC lilac && cd lilac && make
