00001 LILAC: Learning and Integration of Lasers for Adaptive Control
00002 
00003 This project provides a framework with which to analyze/control lasers, and more generally, any tunable dynamical system.
00004 
00005 You can find documentation on the [wiki](http://github.com/schets/LILAC/wiki), and doxygen generated [advanced documentation](http://schets.github.io/LILAC) as well 
00006 
00007 Installation:
00008 This project depends on [FFTW](www.fftw.org), and the makefile has configurations for GCC and the Intel C++ Compiler. These are simple to install on Linux and Macintosh systems and each site provides installation instructions.
00009 
00010 Once you have installed FFTW, you can proceed to download and compile LILAC. LILAC can be downloaded without git by going to the github page and manually downloading the file, but using git as a version control is much more convinient and less error prone.
00011 
00012 On Linux:
00013 
00014 1. From the command line, proceed to the directory in which you want to have lilac installed.
00015 2. Run the command: git clone https://github.com/schets/LILAC lilac
00016 3. Enter the lilac directory (cd lilac)
00017 4. Compile the code by running make
00018 5. Upon successful compilation, the lilac binary can be found in the directory bin
00019 
00020 Or, from to installation directory, copy and paste this into the command line
00021 
00022 git clone https://github.com/schets/LILAC lilac && cd lilac && make
